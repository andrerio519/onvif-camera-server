🛠️ Tech Stack

Aplikasi ini dibangun menggunakan teknologi modern untuk kontrol kamera ONVIF:

🖥️ Backend

  • Node.js - Runtime JavaScript untuk server
  • Express.js - Framework web untuk API
  • node-onvif - Library untuk komunikasi ONVIF
  • fluent-ffmpeg - Converter RTSP ke MJPEG
  • cors - Cross-Origin Resource Sharing

🎨 Frontend

  • Vanilla JavaScript - No framework dependencies
  • Fetch API - Modern HTTP requests
  • CSS3 - Gradients, animations, flexbox & grid
  • Responsive Design - Mobile-friendly interface

📡 Streaming

  • RTSP - Real Time Streaming Protocol
  • MJPEG - Motion JPEG untuk browser
  • FFmpeg - Video transcoding engine
  • TCP Transport - Reliable stream delivery

🎯 Features

  • PTZ Control - Pan, Tilt, Zoom real-time
  • Preset Management - Save & recall positions
  • Live Streaming - MJPEG video feed
  • Camera Info - Device details & profiles
  • Toast Notifications - User feedback system

📖 Panduan Instalasi

1️⃣ Prasyarat

Pastikan software berikut sudah terinstall di sistem Anda:

  • Node.js (v14 atau lebih baru)
  • npm (Node Package Manager)
  • FFmpeg (untuk streaming)

2️⃣ Install FFmpeg

Windows:

# Download dari: https://ffmpeg.org/download.html # Extract dan tambahkan ke PATH

Linux:

sudo apt update sudo apt install ffmpeg

macOS:

brew install ffmpeg

3️⃣ Install Dependencies

Buka terminal di folder project dan jalankan:

npm install express cors body-parser node-onvif fluent-ffmpeg

4️⃣ Jalankan Server

Start backend server dengan command:

node server.js

Server akan berjalan di: http://localhost:3000

5️⃣ Akses Aplikasi

Buka file index.html di browser atau gunakan live server.

Catatan: Pastikan server backend sudah running sebelum mengakses frontend.

6️⃣ Konfigurasi Kamera

Untuk menghubungkan kamera:

  • Masukkan IP Address kamera (contoh: 192.168.1.100)
  • Masukkan Port ONVIF (biasanya 80, 8000, atau 8080)
  • Masukkan Username dan Password kamera
  • Klik tombol "Hubungkan"

🔧 Troubleshooting

Stream tidak muncul:

  • Pastikan FFmpeg sudah terinstall dengan benar
  • Check firewall tidak memblokir port 3000
  • Pastikan kamera support RTSP streaming

Tidak bisa connect:

  • Verifikasi IP address dan credentials benar
  • Pastikan kamera dan komputer dalam network yang sama
  • Test ping ke IP kamera
